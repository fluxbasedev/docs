## v1.0.0 - March 27, 2026

Official 1.0.0 release. 🎉

- Fixed a lot of bugs (listed below) thanks to the alpha testers
- You can now edit a submission (and be added back to the queue) after it has been submitted
- Where applicable, a "verified" checkmark icon will appear for bots whose ownership has been verified

## v0.1.3-alpha - March 16, 2026
- Changed the user section layout to be more consistent
- Fixed the overall page inconsistencies
- Fixed an issue on Chrome where the nav dropdown would cause the layout to shift
- Overhauled the public bot page
- Updated the user sync logic to try prevent 401 Unauthorized errors when failure happens

## v0.1.2-alpha - March 13, 2026
- Overhauled light theme
- Rewritten the API to use hono instead of express for better performance and reliability
- Updated ratelimiting, should see less `Too many requests` errors
- General bug fixes and improvements

## v0.1.1-alpha - March 12, 2026
- Category discovery page
- No longer shows categories that have no bots
- Changed how log messages are sent, so you will now actually receive a ping

## v0.1.0-alpha - March 12, 2026

Initial alpha release. 🎉

### Platform
- Bot listing and discovery
- Bot submission and review queue
- Ownership verification via bio code
- Category browsing
- Bot search

### Dashboard
- Bot onboarding flow
- Submission status tracking
- Bot listing management
- Developer API key management

### API
- Public REST API with API key authentication
- Rate limiting
- OpenAPI documentation via Swagger and Mintlify
